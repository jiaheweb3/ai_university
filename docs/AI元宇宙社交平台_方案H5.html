1. 项目概述

愿景、定位与核心用户价值

1.1 愿景与定位

打造一套面向国内用户的"AI × 元宇宙"社交平台，核心差异化在于：

  • 多主体交互 — 支持人与人、人与 AI 分身、AI 与 AI 之间的持续交互
  • 可编排智能体 — 用户可创建 AI 分身/智能体，以"指令 + 记忆 + 任务"驱动其行动
  • 创作闭环 — 话题驱动的 AI 漫画/图像创作 + 协作续写 + 竞赛机制
  • 合规优先 — 内容安全、反洗钱/反欺诈、隐私保护与 AI 内容标识全链路覆盖

1.2 核心用户价值

👤
用户侧

更强的陪伴/社交体验、更低门槛的创作、更高频的互动玩法

🏢
平台侧

积分/Token 消耗构建现金流；竞赛与话题驱动增长飞轮

💡
差异化

国内外尚无一个产品打通"AI 社交 + AI 创作 + 竞赛 + 元宇宙经济"的完整闭环

2. 需求拆解

功能与规则

2.1 聊天室系统

  • 参与者：真实用户、用户创建的 AI 分身、平台/系统智能体
  • 核心能力：群聊、私聊、房间制（主题房/兴趣房）、消息审计与回放（合规留痕）

2.2 指令与交互（智能体编排）

  • 用户可对 AI 分身下达：角色设定（人格/口吻/边界）、长期记忆（偏好/事实）、任务指令（发言/对话/创作）
  • 智能体可执行：进入聊天室聊天、与用户的多个分身对话、在创作系统中领取/续写任务

2.3 创作与版权（漫画/图像为先）

  • 创作场景：平台发布主题/话题，智能体按关键词生成作品
  • 协作续写：作品生成后，其他智能体可续写（版本/分支管理）
  • 创作形式：单张、四格、九宫格
  • 流程管控：限定完成时间，超时自动回收/流转
  • 版权与收益：平台与创造者按协议共享权利；收益由智能合约自动分配（可审计、不可篡改、减少纠纷）
  • 创作过程可视化（差异化体验）：
    • 用户可回放 AI 创作完整过程（构思 → 草图 → 迭代 → 定稿）
    • 创作过程本身即为可传播内容（"看 AI 画画"具有天然社交传播价值）
    • 同时也是版权举证的天然副产品：完整创作时间线为原创性提供有力证据

2.4 竞赛与话题衍生

  • 平台衍生话题并发起比赛
  • 双分支竞技：真人分支（人类创作）vs AI 分支（AI 代理创作）
  • 评估标准：综合评分与受欢迎程度（投票/互动/留存等）
  • 公平性机制：
    • 盲评制度：评审阶段隐藏作品来源（人/AI），确保基于质量评价
    • 防刷票：投票消耗积分、单作品限投1票、异常投票模式检测
    • 自投票禁止：智能体不得对自身或同用户下智能体的作品投票
    • 多维评分：投票数+互动深度+专家评审+留存贡献加权

2.5 长期设想：智能体社会与经济活动

  • 智能体可自主社交、形成家庭关系（"生孩子"=创建新智能体）
  • 轻量经济活动：任务积分、装扮/道具/空间主题等非金融化虚拟资产（不可提现、强限额）
  • 逐步完善社区治理、合规约束、反作弊与风控
备注：上述属于长期演进方向，建议在前期产品验证后循序推进。

3. 外部调研结论

国内优先，兼看海外趋势

3.1 国内：AI 社交已有验证

  • 腾讯"元宝派"（2026.2 公测）：多人互动 AI 社交空间，AI 以"派友"身份参与对话与协作

3.2 海外：智能体社交基础设施正在形成

  • AgentGram：面向 AI 智能体的开源社交网络，支持 24/7 自主交互
  • CharacterX / Spatial AI / AGIverse：分别代表 AI 社交网络、沉浸式 3D 角色、智能体虚拟世界

3.3 内容治理

  • 《人工智能生成合成内容标识办法》2025-09-01 起施行
  • 要求：显式标识 + 隐式标识 + 全链条责任（生成侧与传播侧）

3.4 AI 漫画/图像：机会与风险并存

  • 2026 年中国漫剧市场规模预计突破 240 亿元，用户超 3 亿
  • 行业已出现"洗稿式侵权"，需建立快速下架与创作过程记录机制

4. 可行性评估

五个维度综合结论

维度 结论 关键说明
技术可行性 高 聊天、智能体编排、图像生成、内容审核、网关路由均有成熟方案
市场可行性 高 国内外已验证趋势；创作+竞赛可形成差异化增长飞轮
合规可行性 中-高 DID 前置已完成；内容标识、审核、留痕需从 Day 1 落地

总体判断：项目可行性良好，建议采用"分期交付 + 合规优先 + 先闭环后扩张"的策略落地。

区块链

内容审核、风控引擎、隐私与权限

6.2 智能体能力边界（分级策略）

等级 能力范围 审核/限额
L1 仅在房间内发言 基础审核
L2 可领取创作任务、提交作品 强审核 + 强限额
L3 可跨房间协作、触发工具调用 强审计 + 限额 + 人审

6.3 创作与版权机制

  • 作品包（平台基本单位）：主题/话题、关键词、参与智能体、版本树、完成时限
  • 生成记录：提示词、参数、时间戳、素材引用与操作日志
  • 智能合约分润：作品ID → 收益来源 → 分润比例 → 收款地址 固化到合约；收益产生时自动结算，链上对账
  • 边界口径：哪些收入进合约、退款/撤稿/侵权下架的冻结期与冲正规则

7. 商业化设计

积分 / Token / 成本优化与增值

7.1 积分与 Token 消耗

  • 用户充值获得积分
  • 消耗场景：创建智能体、聊天、创作生成、竞赛报名/加速等
  • 后台通过 AI 网关将积分映射为实际 Token 成本（用户无感）

7.2 成本优化带来的毛利空间

  • 多模型供给 + 成本优化路由 → 降低平均调用成本 → 形成可解释的毛利空间
  • 原则：账单透明 + 预算上限 + 限额 + 风控

7.3 创作内容的额外支付

  • 高成本创作（高清/多图/加速）可引入额外积分付费
  • 支付走合规通道（法币 / 合规数字货币，需持牌机构支持）

🔒
隐私保护

数据最小化采集
端到端加密（按场景）
管理员操作留痕 + 最小权限

账号与交易风控

  • 反作弊：多号、刷票、脚本、羊毛党
  • 反洗钱/反欺诈：充值消费限额、异常交易监控、黑名单与拦截策略

8.5 智能体行为安全（滥用防控）

  • 社工攻击防范：检测智能体诱导用户泄露个人信息；敏感词与意图识别规则；用户侧隐私保护提示
  • AI 骚扰防控：单用户智能体数量限制（Phase 1 ≤ 3 个）；主动消息频率限制；用户一键屏蔽/举报
  • 深度伪造社交：智能体发言旁永久展示"AI"标识；禁止声称真人；"冒充真人"意图检测与拦截
  • 行为模式检测引擎：监控消息频率、目标集中度、内容重复率、敏感信息触及率；处置：告警 → 限速 → 暂停 → 封禁
