# 项目决策记录

> 项目：AI × 元宇宙社交平台
> 创建时间：2026-03-15

## 项目定位
- **产品方向**：面向国内用户的 "AI × 元宇宙" 社交平台
- **核心差异化**：多主体交互（人-人 / 人-AI / AI-AI）、可编排智能体、AI 创作闭环、合规优先
- **前置条件**：实名 DID（"前台匿名、后台实名"）已完成接入

## 分期策略
- **Phase 1 MVP**：社交 + 智能体基础（方案 A: 8-10 周 / 方案 B: 3-4 个月）
- **Phase 2**：创作闭环与商业化基础（3-4 个月）
- **Phase 3**：竞赛与话题系统（2-3 个月）
- **Phase 4**：元宇宙社会与经济系统（第 12 个月起持续迭代）

## 关键设计决策
1. **智能体分级**：L1（仅发言）→ L2（创作任务）→ L3（跨房间协作）
2. **版权机制**：智能合约自动分润（作品ID → 收益来源 → 分润比例 → 收款地址）
3. **商业模式**：积分/Token 消耗 + 多模型路由成本优化 + 创作增值支付
4. **合规**：AIGC 显式/隐式标识全链路、内容审核（同步拦截 + 异步下架）、留痕 ≥ 6 个月
5. **Phase 1 明确不做**：可提现/金融化、跨房间自治、智能体家庭/高敏叙事、多维宇宙、虚实商业、AI结婚生育

## 2026-03-18 新增决策
6. **核心产品哲学**：功能是"长出来的"，不是"设计出来的"。多维宇宙等远景 ≥ 5 年后再考虑，Phase 1 只验证核心假设
7. **外部 Agent 接入**：协议无关（不绑定 OpenClaw 或特定平台），平台提供标准 Agent Gateway API，国内 Agent 平台优先接入。Phase 1A 不开放，Phase 1B 邀请制开放
8. **可行性评估**：综合评分 8/10。AI 分身社交需求已验证（Soul 1100万DAU / Character.AI 2800万MAU），"Agent 社交网络"为无人占据的全新品类
9. **投资人叙事调整**：主打"AI 分身社交"而非"元宇宙"，远景提"AI Agent 社交网络"
10. **Go-No-Go 框架**：MVP 上线 + 30 天决策，核心指标：D7 留存 ≥ 8%、日均发言率 ≥ 20% DAU、积分付费转化 ≥ 3% MAU
11. **AI 驱动开发**：全部代码由 AI Agent（恺撒）编写，创始人担任 PM + 协调员。无人力薪资支出，外包 UI 设计 + 安全审计 + 法务合规。Phase 1 预算从 ¥127 万降至 ¥9-14 万
12. **多 Agent 并行开发**：同时运行 2-3 个 AI Agent 对话窗口，分别负责后端核心、AI 引擎、Flutter 前端。前提是 Phase 0 先定好 API Schema + DB Schema + 共享类型作为契约
13. **外部 Agent 接入提前至 Phase 1**：作为差异化核心竞争力，Phase 1 即支持外部 Agent/分身接入。协议无关（AAP）、安全优先、平台网关模式、国内优先。首批邀请制 5-10 个合作伙伴

## 技术约束
- 消息：先实时通道再落审计库，Outbox 模式保证至少一次
- 智能体：队列化执行、幂等键、单体熔断隔离
- 记忆：Phase 1 仅短期上下文 + 可控摘要，用户可见可删
- 多模型路由：离线评测 + 线上 AB + 单日预算上限
