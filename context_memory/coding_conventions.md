# 编码规范

> 项目：AI × 元宇宙社交平台
> 创建时间：2026-03-15

## 项目状态
- 当前处于**方案规划阶段**，尚未进入代码开发
- 项目文档存放在 `docs/` 目录下

## 文档规范
- 主方案文档使用 Markdown 格式，带 YAML frontmatter
- 演示/汇报使用独立 H5 页面（深色主题，glassmorphism 风格）
- 汇报版本使用 docx 格式，已迭代至 v4

## H5 演示页面设计规范
- **配色**：深色系（`--primary: #0a1628`），金色强调（`--accent: #c9a34f`）
- **字体**：PingFang SC / Microsoft YaHei / Helvetica Neue
- **组件**：卡片（`.card`）、时间轴（`.timeline`）、标签（`.tag`）、流程图（`.flow`）
- **动画**：IntersectionObserver 触发的滚动渐现（`.reveal`）
- **响应式**：640px 断点适配移动端
